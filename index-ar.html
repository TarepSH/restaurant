مطعم الـ CSS

لا تقلق، نحن هنا لمساعدتك!

ستقوم بتعلم استخدام المنتقي في الـ CSS المنتقي هو الذي يحدد العناصر في الـ HTML التي تريد ان تقوم بالتعديل على شكلها.

الفصل الـ 1 - قاعدة الـ CSS

p {
   margin-bottom: 12px;
}

هنا, الـ "p" هو المنتقي (قام بأختيار كل <p> العناصر) وسيقوم بتطبيق خاصية margin-bottom عليهم.

لتبدأ في اللعبة، قم بكتابة الـ CSS والمنتقي ضمن المحرر النصوص في الاسفل، وقم بأختيار العنصر الصحيح من الطاولة الطعام. اذا كان جوابك صحيح، سيتم نقلك إلى التحدي التالي.

قم بتمرير مؤشر الفأرة على العناصر في الطاولة، لتشاهد تخطيطهم في الـ HTML
